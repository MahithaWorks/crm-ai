# CRM AI Dashboard

A modern AI-powered CRM (Customer Relationship Management) dashboard designed to manage leads, track sales pipelines, schedule visits, and generate smart insights.

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 👥 Lead Management (Add, View, Prioritize Leads)
- 🔄 Pipeline Tracking (Lead Progress Flow)
- 📅 Visit Scheduling with Reminders
- 📊 Dashboard Analytics (Charts & Stats)
- 🤖 AI Insights (Smart Recommendations)
- 🧑‍💼 Agent Performance Tracking
- 🌙 Dark Theme UI (Modern Design)

## 🧠 Smart Insights

- Detects high-value leads
- Suggests follow-up urgency
- Highlights lead quality trends

## 📊 Dashboard Highlights

- Lead Distribution Chart
- Agent Performance Chart
- Top Agent Identification
- Real-time analytics updates

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- Recharts (for charts)
- LocalStorage (for data persistence)
